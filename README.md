# Recipe App
Designed a simple recipe website. You can add, delete, sort and filter the recipes.

# Used Technologies
- HTML
- CSS
- JS
- React
- react-router-dom
- Axios
- tailwind
- react-select
- react-icons
- react-toastify
- express.js
